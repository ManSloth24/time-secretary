Time Secretary

Time Secretary is a private, self-hosted productivity, time-tracking, reminder, work-hours, and local dashboard notification system.

Product And Messaging Use Case

Time Secretary sends operational SMS messages only. Messages may include:

  • Scheduled check-ins.
  • Activity logging prompts.
  • Reminders.
  • Command responses.
  • Work-hours and location confirmations.
  • Daily summary notifications.
  • Link-only local dashboard notifications.

Time Secretary does not send marketing or promotional SMS messages.

Full reports, secure captures, work notes, run notes, process notes, and briefing contents are stored locally and are not sent by SMS by default. SMS may contain only operational responses or a link to the user's local/private dashboard.

This page is public so SMS reviewers can understand the product and messaging use case without logging in.

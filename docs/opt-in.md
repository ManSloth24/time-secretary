# Time Secretary SMS Opt-In

Effective date: June 29, 2026

## Program Name

Time Secretary

## Program Description

Time Secretary sends private operational SMS messages for productivity and time tracking, including:

- Scheduled time check-ins.
- Activity logging prompts.
- Reminders.
- Command responses.
- Work-hours and location confirmations.
- Daily summary notifications.
- Link-only local dashboard, report, and briefing notifications.

Time Secretary does not send marketing or promotional SMS messages.

## Consent Workflow

1. The user visits this Time Secretary SMS Opt-In page.
2. The user reviews the SMS program description, [Privacy Policy](privacy-policy.md), and [Terms & Conditions](terms-and-conditions.md).
3. The user enters their mobile phone number into the Time Secretary local app or configuration.
4. The user actively consents to receive recurring SMS messages from Time Secretary.
5. The user may initiate messaging by texting the Time Secretary number or by enabling scheduled reminders/check-ins in the local app.
6. The user can opt out at any time by replying STOP.
7. The user can request help by replying HELP.

Because Time Secretary is a private self-hosted app, opt-in is controlled by the app owner/operator through the local application or configuration rather than a public multi-user signup form.

## SMS Consent Language

By opting in, I agree to receive recurring SMS messages from Time Secretary, including time-tracking check-ins, reminders, work-hours/location confirmations, command responses, and link-only local dashboard notifications. Message frequency varies based on my settings and may be as often as every 15 minutes during configured active hours. Message and data rates may apply. Reply STOP to opt out. Reply HELP for help.

## Message Frequency

Message frequency varies based on user settings. If 15-minute check-ins are enabled, messages may be sent as often as every 15 minutes during configured active hours.

## Message And Data Rates

Message and data rates may apply.

## STOP / HELP

Reply STOP to opt out. Reply HELP for help.

## Sensitive Data

Sensitive work notes, secure captures, run notes, process details, recipe/program changes, and briefing/report contents are intended to remain in the local application and are not sent by SMS by default.

## Sample Opt-In Confirmation

Time Secretary: You are opted in to receive recurring productivity, time-tracking, reminder, work-hours/location, command response, and link-only local dashboard notification messages. Message frequency varies. Message and data rates may apply. Reply HELP for help or STOP to opt out.
